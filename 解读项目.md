## 🎯 项目需要跑什么

### 主要任务
**Alpamayo 1** 是一个用于自动驾驶的视觉-语言-动作（Vision-Language-Action）模型，主要功能是：

1. **轨迹预测**：预测车辆未来 6.4 秒的轨迹（64 个航点，10 Hz）
2. **因果链推理（Chain-of-Causation）**：解释为什么采取特定驾驶动作
3. **多摄像头视觉理解**：处理多个摄像头的视频输入

### 运行方式
- **测试脚本**：`python src/alpamayo_r1/test_inference.py`
- **交互式笔记本**：`notebook/inference.ipynb`

## 📥 项目输入

| 输入类型 | 具体内容 | 说明 |
|---------|---------|------|
| **多摄像头视频** | 4 个摄像头的图像序列 | - 前广角（120°FOV）<br>- 前长焦（30°FOV）<br>- 左侧交叉（120°FOV）<br>- 右侧交叉（120°FOV） |
| **自运动历史** | 过去 1.6 秒的车辆运动 | 位置（xyz）和旋转（rotation），16 个时间步，10 Hz |
| **时间戳** | 相对和绝对时间戳 | 用于对齐多摄像头数据 |

### 输入示例
从 `test_inference.py` 可以看到：
```python
# 输入数据结构
{
    "image_frames": (N_cameras, num_frames, 3, H, W),  # 多摄像头图像
    "ego_history_xyz": (1, 1, num_history_steps, 3),   # 历史位置
    "ego_history_rot": (1, 1, num_history_steps, 3, 3), # 历史旋转
    ...
}
```

## 📤 项目输出

| 输出类型 | 具体内容 | 说明 |
|---------|---------|------|
| **预测轨迹** | 未来 6.4 秒的车辆轨迹 | xyz 坐标，64 个航点，10 Hz |
| **因果链推理** | 自然语言解释 | 解释为什么采取特定驾驶动作 |
| **旋转预测** | 未来车辆姿态 | 旋转矩阵（3x3） |

### 输出示例
从运行结果可以看到：
```
Chain-of-Causation (per trajectory):
 [['Nudge to the left to increase clearance from the construction cones encroaching into the lane.']]
```

**轨迹输出**：
- `pred_xyz`: 预测的 3D 位置坐标
- `pred_rot`: 预测的旋转矩阵

## 🏗️ 架构特点

### 模型组成
1. **Cosmos-Reason 主干网络**：视觉-语言模型，负责推理
2. **动作专家模块**：专门处理轨迹预测
3. **扩散模型**：生成多样化的轨迹样本

### 核心特性
- ✅ Chain-of-Causation 推理（混合自动标注 + 人工干预）
- ✅ Vision-Language-Action 架构
- ✅ 6.4 秒轨迹预测
- ❌ 不包含 RL 后训练（当前版本）
- ❌ 不包含路线/导航条件（当前版本）
- ❌ 不包含元动作/通用 VQA（当前版本）

## 📊 性能评估

- **minADE**：最小平均位移误差（你的运行结果：2.56 米）
- 用于衡量预测轨迹与真实轨迹之间的距离
